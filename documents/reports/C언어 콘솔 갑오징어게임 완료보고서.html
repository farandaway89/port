7. 발견된 문제점 및 해결 과정

⚠️ 주요 문제점 1: 실시간 입력 처리의 한계

현상: 무궁화꽃 게임에서 실시간 키 입력 감지가 어려워 게임의 긴장감 저하

원인: C언어 표준 입출력의 블로킹 특성과 콘솔 환경의 제약

💡 해결 과정:

• ncurses 라이브러리 검토했으나 복잡도로 인해 보류
• 턴 기반 입력 방식으로 게임 방식 수정
• sleep() 함수와 조합하여 시간 제한 효과 구현

⚠️ 주요 문제점 2: 복잡한 ASCII 아트 구현 어려움

현상: 시각적으로 매력적인 게임 화면 구성이 예상보다 어려움

원인: 콘솔 출력의 제한과 ASCII 아트 설계 경험 부족

💡 해결 과정:

• 복잡한 그래픽 대신 간단하고 직관적인 텍스트 UI 채택
• 기능성에 중점을 두고 화려함보다 명확성 우선
• 컬러 코드 활용으로 최소한의 시각적 효과 추가

⚠️ 주요 문제점 3: 팀원 간 코드 통합 충돌

현상: 각자 작성한 함수들을 합칠 때 변수명과 함수 인터페이스 불일치

원인: 사전 코딩 규칙 정의 부족과 통합 계획 미흡

💡 해결 과정:

• 팀장 주도로 변수명과 함수명 통일 작업 수행
• 공통 헤더 파일 작성으로 인터페이스 표준화
• 일일 코드 리뷰를 통한 조기 문제 발견
