🔧 기술적 구현 세부사항

프론트엔드 아키텍처

  • React 18: Hooks 기반 함수형 컴포넌트, 실시간 상태 관리
  • Vite: 개발 서버, HMR(Hot Module Replacement) 지원
  • Tailwind CSS: 유틸리티 기반 스타일링, 반응형 디자인
  • Chart.js: 실시간 뇌파 데이터 시각화 라이브러리

데이터 처리 로직

  • 시뮬레이션 엔진: Math.random() 기반 EEG 데이터 생성
  • 상태 관리: React useState, useEffect를 통한 실시간 업데이트
  • 차트 렌더링: 1초 간격 데이터 업데이트, 20개 포인트 슬라이딩 윈도우
  • 모듈화: 3개 독립 컴포넌트 (RealtimeMonitor, AdvancedAnalytics, ClinicalReport)

배포 및 성능

  • Vercel 배포: 자동 빌드, CDN 배포, HTTPS 지원
  • 번들 최적화: 372KB 번들 크기, Tree-shaking 적용
  • 반응형 지원: 모바일/태블릿/데스크톱 최적화
  • 접근성: 다크모드 지원, 시맨틱 HTML 구조

⚠️ 프로젝트 한계 및 제약사항

명확한 한계 사항

🔴 실제 EEG 하드웨어 미연결

OpenBCI나 실제 EEG 장비와 연결되지 않으며, 모든 데이터는 JavaScript로 생성된 시뮬레이션입니다.

🔴 AI 모델 부재

TensorFlow.js를 import했지만 실제 훈련된 감정 분류 모델은 없으며, 감정 분류는 랜덤 생성입니다.

🔴 의료 정확성 없음

의료 기기 인증이나 임상 검증을 거치지 않았으며, 교육/시연 목적의 프로토타입입니다.

🔴 데이터 지속성 없음

모든 데이터는 브라우저 메모리에만 저장되며, 새로고침 시 모든 데이터가 소실됩니다.
