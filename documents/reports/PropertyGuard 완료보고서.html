4. 🔧 기술적 구현 내용

✅ 성공적으로 구현된 기술

Qt/C++ 데스크톱 애플리케이션

Qt 6.0을 활용한 현대적인 GUI 구현. Model-View 아키텍처 적용으로 데이터와 UI의 분리를 달성했습니다.

MySQL 데이터베이스 연동

Qt SQL 모듈을 통한 안정적인 데이터베이스 연결. 트랜잭션 처리와 데이터 무결성을 보장합니다.

가격 이상치 탐지 알고리즘

통계적 방법을 사용한 가격 이상치 탐지. 지역별 평균가 대비 편차를 계산하여 의심스러운 매물을 식별합니다.

⚠️ 제약사항 및 타협점

머신러닝 모델 간소화

원래 계획했던 복잡한 ML 모델 대신 통계적 분석 방법을 채택했습니다. 시간 제약과 데이터 부족으로 인한 현실적 선택이었습니다.

자연어 처리 기능 축소

고급 NLP 대신 키워드 매칭 방식을 사용했습니다. "급매", "헐값" 등 미리 정의된 패턴만 탐지 가능합니다.

실시간 처리 최적화 미완

대용량 데이터 처리 시 성능 이슈가 있습니다. 인덱싱과 캐싱 최적화가 추가로 필요합니다.
