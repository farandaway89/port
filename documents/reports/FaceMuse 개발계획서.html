📋 프로젝트 개요

🎯 프로젝트 목적

FaceMuse는 TensorFlow.js와 BlazeFace 모델을 활용하여 실시간으로 얼굴 특징을 분석하고, 사용자의 개성과 특성을 종합적으로 진단하는 웹 애플리케이션입니다.

🌟 핵심 가치

  • 단순한 동물상 분류를 넘어선 상세한 얼굴 특징 분석
  • 성격, 직업 적성, 라이프스타일을 포함한 종합적 개성 진단
  • 20개 이상의 창의적 분석으로 사용자 흥미 극대화
  • 성별별 맞춤 분석으로 개인화된 결과 제공

⏰ 개발 일정

Phase 1 (2시간): 프로젝트 초기 설정
- Next.js 15 프로젝트 생성
- TensorFlow.js 및 BlazeFace 모델 설정
- 기본 UI 레이아웃 구성
Phase 2 (4시간): 얼굴 인식 및 분석 로직
- 이미지 업로드 기능 구현
- 얼굴 특징점 추출 및 분석
- 얼굴상 분류 알고리즘 구현
Phase 3 (4시간): 종합 분석 시스템
- 성격 분석 및 직업 추천 로직
- 20개 창의적 분석 기능 구현
- 성별별 맞춤 분석 적용
Phase 4 (6시간): UI/UX 및 최적화
- 반응형 디자인 완성
- 애니메이션 및 인터랙션 추가
- TypeScript 타입 안정성 확보
- Vercel 배포 및 최종 테스트
