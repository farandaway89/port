AI 이미지 프롬프트 생성기 개발계획서

프로젝트 기간: 2025/8/1(목) ~ 2025/8/5(월) (5일간)

작성일: 2025년 8월 1일

개발자: 이승필

1. 프로젝트 개요

🎯 개발 목적

AI 이미지 생성을 위한 창의적이고 구체적인 프롬프트를 자동으로 생성하는 웹 기반 도구 개발. 사용자가 간단한 옵션 선택으로 전문적인 AI 이미지 생성 프롬프트를 만들 수 있도록 지원

🏗️ 개발 환경

  • Framework: Gradio 4.0+
  • Language: Python 3.7+
  • Interface: Web-based GUI
  • Deployment: Hugging Face Spaces
  • Storage: 메모리 기반 (세션 관리)
  • Theme: Soft Theme (Indigo/Purple)

📋 주요 특징

  • 직관적인 드롭다운 기반 인터페이스
  • 10가지 이미지 스타일 지원
  • 9가지 분위기/무드 옵션
  • 6개 프리셋 템플릿 제공
  • 실시간 프롬프트 생성 및 복사
  • 랜덤 생성 기능
  • 통계 정보 표시

⚠️ 기술적 제약사항

  • 인터넷 연결 필요 (Gradio 웹앱)
  • 세션 기반 데이터 (영구 저장 없음)
  • Hugging Face Spaces 리소스 제한
  • 실시간 이미지 생성 기능 없음

4. 기술 아키텍처

  • Python Standard Library: 문자열 처리
  • Random Module: 랜덤 생성
  • JSON Module: 데이터 구조 관리

🎨 프론트엔드 구성

  • Gradio Blocks: 커스텀 레이아웃

  • COLORS: 9가지 색상 팔레트
  • COMPOSITIONS: 8가지 구도
  • TEMPLATES: 6개 프리셋

🚀 배포 환경

6. 프롬프트 데이터베이스 설계

📚 스타일 카테고리

  • 사실적: photorealistic, ultra realistic
  • 예술적: oil painting, watercolor
  • 디지털: digital art, 3D render
  • 만화: anime, comic book style

🎭 분위기 매핑

  • 밝은: bright, cheerful, vibrant
  • 어두운: dark, mysterious, moody
  • 환상적: dreamy, ethereal, magical
  • 미래적: futuristic, sci-fi, cyberpunk

🌈 색상 키워드

  • 따뜻한: warm colors, golden hour
  • 차가운: cool colors, blue tones
  • 생생한: vibrant, saturated, bright
  • 파스텔: pastel colors, soft tones

📐 구도 용어

  • 거리: close-up, medium shot, wide shot
  • 각도: low angle, high angle, bird's eye
  • 시점: first person, panoramic view
