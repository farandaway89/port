구현된 핵심 기능

TCP 네트워크 통신: System.Net.Sockets를 사용한 서버-클라이언트 간 안정적인 연결 및 패킷 교환 시스템 구현
WinForm GUI: 사용자 친화적인 채팅 인터페이스와 Invoke/BeginInvoke를 활용한 UI 스레드 안전성 보장
다중 사용자 지원: TcpListener를 활용한 동시 다중 클라이언트 연결 관리 및 멀티스레딩 구조 구현
실시간 메시지 전송: 대리자/이벤트 시스템을 통한 실시간 메시지 브로드캐스트 기능 완성
사용자 상태 관리: 접속/퇴장 상태를 실시간으로 추적하고 다른 사용자에게 알림

추가 구현된 기능

메시지 히스토리: ListView를 활용한 채팅 메시지 리스트와 자동 스크롤 기능
사용자 목록: 현재 접속 중인 사용자 목록을 실시간으로 표시하는 기능
연결 상태 표시: 서버 연결 상태를 시각적으로 표시하는 상태바 구현
오류 처리: 네트워크 오류, 연결 실패 등에 대한 사용자 친화적 오류 메시지 표시
메시지 포맷팅: 발신자, 시간, 메시지 내용을 구분하여 표시하는 포맷팅 시스템
2. 기술적 성과 및 학습 결과

적용된 기술 스택

C# WinForm TCP Socket 비동기 프로그래밍 멀티스레딩 대리자/이벤트 네트워크 프로그래밍 UI 스레드 안전성 실시간 통신

핵심 기술 습득

  • TCP 소켓 프로그래밍: TcpListener, TcpClient를 활용한 네트워크 통신 구현
  • 비동기 프로그래밍: async/await 패턴을 통한 논블로킹 네트워크 작업
  • 멀티스레딩: 다중 클라이언트 동시 처리 및 스레드 안전성 확보
  • 대리자/이벤트: 콜백 시스템을 통한 실시간 이벤트 처리
  • UI 스레드 안전성: Invoke/BeginInvoke를 통한 크로스 스레드 UI 업데이트

프로그래밍 패턴 학습

  • Observer 패턴: 이벤트 기반 메시지 전파 시스템
  • Producer-Consumer 패턴: 메시지 큐를 통한 비동기 처리
  • Singleton 패턴: 서버 인스턴스 관리
  • Command 패턴: 사용자 입력 처리 및 명령 실행
  • Factory 패턴: 클라이언트 연결 객체 생성
